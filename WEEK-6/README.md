# WEEK-6 Projects

## Overview
This workspace contains a small web development project with five hands-on HTML/CSS practice assignments. Each folder (`Handson-1` through `Handson-5`) contains a standalone lesson demonstrating basic web page layout, styling, and responsive design principles.

## Project Structure
- `Handson-1/` - First practice lesson with `index.html` and `styles.css`
- `Handson-2/` - Second practice lesson with `index.html` and `style.css`
- `Handson-3/` - Third practice lesson with `index.html` and `style.css`
- `Handson-4/` - Fourth practice lesson with `index.html` and `style.css`
- `Handson-5/` - Fifth practice lesson with `index.html` and `styles.css`
- `assests/` - Image asset folder used by one or more practice pages

## How to View
Open any `index.html` file in a web browser. For the best result, use a modern browser such as Microsoft Edge, Google Chrome, or Firefox.

## Notes
- Each `Handson` folder is self-contained and can be opened independently.
- File names for CSS vary slightly between lessons (`style.css`, `styles.css`).
- The page content and layout style may differ between practice files.

## Tips
- Use the browser developer tools to inspect layout and CSS rules.
- Compare the different `Handson` folders to see how styling and HTML structure evolve.
- Keep practicing by modifying the HTML and CSS to learn how each change affects the page.
