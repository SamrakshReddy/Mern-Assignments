# Week 2 - JavaScript Data Handling and DOM Concepts

This folder contains the Week 2 assignment examples focused on data copying, date handling, and a simple todo app structure.

## Files

- `date-demo.js` - Demonstrates working with JavaScript `Date` objects, formatting, and time values.
- `deepcopy.js` - Shows how deep copying works for nested objects and arrays.
- `shallowcopy.js` - Explains shallow copying behavior and how references are handled.
- `todo/` - Contains a small todo application example separated into modular files.
  - `todo/app.js` - Main todo application logic.
  - `todo/task.js` - Defines task creation and task-related behavior.
  - `todo/validate.js` - Handles validation logic for todo input and state.

## How to run

1. Open a terminal in this folder.
2. Run a file with Node.js:

```powershell
node <filename>.js
```

For example:

```powershell
node date-demo.js
```

## Notes

- `date-demo.js` is useful for learning how JavaScript handles dates, timestamps, and formatting.
- `deepcopy.js` and `shallowcopy.js` compare copy techniques so you can see how nested object references behave.
- The `todo` folder organizes a basic app structure to demonstrate modular JavaScript and input validation.
- Modify the files and rerun them to observe changes in behavior or to add new functionality.
