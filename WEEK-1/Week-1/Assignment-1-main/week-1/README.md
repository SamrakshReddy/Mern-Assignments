# Week 1 - JavaScript Fundamentals

This folder contains the Week 1 assignment examples for basic JavaScript concepts.

## Files

- `arrayadv.js` - Advanced array methods and array handling patterns.
- `arraysmple.js` - Simple array operations and examples.
- `classDemo.js` - Demonstrates JavaScript classes and object-oriented syntax.
- `collections.js` - Shows working with collection types such as arrays and objects.
- `control.js` - Control flow examples using conditionals and loops.
- `object.js` - Object creation, property access, and manipulation.
- `operators.js` - Uses JavaScript operators, expressions, and comparisons.

## How to run

1. Open a terminal in this folder.
2. Run a file with Node.js:

```powershell
node <filename>.js
```

For example:

```powershell
node control.js
```

## Notes

- These examples are intended for learning and practicing JavaScript fundamentals.
- Modify the files and rerun them to test different behaviors.
- Use `console.log()` output to verify how each operation works.
