# WEEK-7

Week 7 contains React projects built with Vite. Each project has its own `package.json`, source files, and development scripts.

## Projects

- `React-Productapp-demo` - Product listing application with React Router, reusable layout components, and Tailwind CSS.
- `country-explorer` - Country explorer application with search and country card/list components.
- `task-manager` - Task manager application with add-task form, task list, and task item components.

## Tech Stack

- React
- Vite
- Tailwind CSS
- ESLint
- React Router, used in the product app
- React Hook Form, used in the task manager

## How to Run

Open a terminal in any project folder, then run:

```bash
npm install
npm run dev
```

For example:

```bash
cd WEEK-7/task-manager
npm install
npm run dev
```

## Available Scripts

Most Week 7 projects include:

- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run lint` - run ESLint checks
- `npm run preview` - preview the production build
