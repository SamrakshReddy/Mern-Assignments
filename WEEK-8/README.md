# WEEK-8

Week 8 contains the setup for a user management MERN assignment split into backend and frontend folders.

## Folder Structure

- `BackEnd` - Node.js backend package setup using Express, Mongoose, and dotenv.
- `FrontEnd` - Frontend package setup with Tailwind CSS and Vite-related dependencies.

## Backend

The backend package is configured as an ES module project.

Dependencies:

- Express
- Mongoose
- dotenv

To install backend dependencies:

```bash
cd WEEK-8/BackEnd
npm install
```

## Frontend

The frontend package includes Tailwind CSS dependencies.

To install frontend dependencies:

```bash
cd WEEK-8/FrontEnd
npm install
```

## Notes

- `node_modules` is ignored and should not be committed.
- Add environment variables in a local `.env` file when required by the backend.
- Run backend and frontend commands from their respective folders.
